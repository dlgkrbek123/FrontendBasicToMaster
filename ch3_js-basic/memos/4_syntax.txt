expression
- 값으로 평가되는 syntax

statement
- expressions이나 다른 statement를 조합해서 동작을 지시

---------------------------------------------------------------------

아래의 항목들은 falsy하다고 판단된다.
- false
- ''
- 0 / NaN
- undefined / null

---------------------------------------------------------------------

instanceof 연산자
- 객체의 클래스를 확인
- 프로토타입 체인으로 판단
> car instanceof Vehicle


in 연산자
- 객체가 프로퍼티를 가지는지 확인
> name in Obj


typeof 연산자
- 피연산자의 타입을 문자열로 리턴
- 프리미티브 타입을 구분하는데 유용
- 객체타입은 정확하게 얻을 수 없음
