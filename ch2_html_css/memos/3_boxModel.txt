각 요소는 CSS 박스모델에 맞춰 렌더링
- content, padding, border, margin으로 구성
- box-sizing과 연동
- 요소의 너비와 높이를 계산하는 방식을 지정

---------------------------------------------------------------------

여백 상쇄
- 인접한 요소의 상하여백이 겹치면 하나로 합쳐지는 현상
- margin은 여백 상쇄때문에 width와 height에 미포함
- 더 큰 여백으로 상쇄됨
