CSS : Cascading Style Sheet
- 레이아웃과 스타일 

selector {
  property: value;
}

- 부모의 스타일의 일부는 자식에게 상속되어 전달된다.
- 자동으로 상속되지 않는 스타일은 inherit로 명시적으로 상속

---------------------------------------------------------------------

3가지 방법으로 적용

1. <link rel="stylesheet" href="링크" />로 연결
2. <style></style> 활용
3. 요소에 직접 적용

---------------------------------------------------------------------

selector: 스타일 적용할 대상을 지정
- 전체, 태그명, id, class, 속성 선택자등이 있다.
- ,를 통해 여러 선택자에 동일한 스타일을 지정

* {}
div {}
#a {}
.side {}
a[href="https://example.com"]

---------------------------------------------------------------------

우선순위와 명시도
- 요소에 여러 선택자에 의해 스타일이 적용되면 어려움이 생김
- 적용되는 스타일간의 우선순위를 알아보자

1. inline style이나 !important
2. 명시도가 높은 스타일이 적용
 - A-B-C의 조합으로 계산 (각각 개수를 의미)
 - 앞 단계 숫자가 큰놈이 이김

3. 같은 스타일 적용부에 대해서는 나중에 작성한 스타일이 우선
